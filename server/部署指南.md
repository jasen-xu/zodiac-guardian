# AI 智能解卦后端部署指南

## 前置条件
- Node.js 18+ (推荐使用 LTS 版本)
- 通义千问 API Key（在 https://dashscope.console.aliyun.com/apiKey 申请）

## 本地运行

### 1. 安装依赖
```bash
cd server
npm install
```

### 2. 配置环境变量
```bash
cp .env.example .env
# 编辑 .env 文件，填入你的通义千问 API Key
```

### 3. 启动服务
```bash
npm start
# 服务默认运行在 http://localhost:3000
```

### 4. 访问网站
打开 http://localhost:3000 即可同时访问前端页面和 AI 解卦功能

## 云端部署

### 阿里云/腾讯云部署
1. 购买云服务器（ECS/CVM），安装 Node.js 18+
2. 上传 server 目录到服务器
3. 安装依赖: `npm install --production`
4. 创建 .env 文件并配置 API Key
5. 使用 PM2 守护进程:
```bash
npm install -g pm2
pm2 start server.js --name zodiac-guardian
pm2 save
pm2 startup
```

### 环境变量说明
| 变量 | 必填 | 说明 |
|------|------|------|
| DASHSCOPE_API_KEY | 是 | 通义千问 API Key |
| MODEL | 否 | AI模型，默认 qwen-plus，可选 qwen-turbo / qwen-max |
| PORT | 否 | 服务端口，默认 3000 |

### Nginx 反向代理配置（可选）
```nginx
server {
    listen 80;
    server_name your-domain.com;
    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
